Installing pageit
=================

Using pip
---------
The fastest way to get pageit is using ``pip``:

.. code-block:: bash

    $ pip install pageit

From Source
-----------
You can also install pageit from source:

.. code-block:: bash

    $ git clone git://github.com/metaist/pageit.git
    $ pip install -r requirements.txt --use-mirrors
    $ [sudo] paver install
